# simple calculator cli
## Description
A command-line calculator developed in Python that performs basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Error Handling
- User Input Validation

## Technologies Used
- Python 3

## How to Run

1. Clone the repository
2. Open terminal
3. Run:

python calculator.py

## Sample Output

===== Simple Calculator =====
Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5
